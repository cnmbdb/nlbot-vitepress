# 使用手册

作为管理员，您可以直接在 Telegram 中向机器人发送以下指令进行管理：

- `/admin` - 呼出管理面板
- `/add_goods` - 快速添加商品
- `/order_list` - 查看最近订单
- `/broadcast` - 发送全员广播

### 独角数卡后台
您可以继续使用独角数卡的原生后台管理商品、卡密和支付接口。所有更改会实时同步到机器人。

### 机器人可视化后台
我们还为您提供了专属的机器人 Web 后台，专注于 Telegram 特有的功能配置：

- **菜单配置**：自定义机器人的底部键盘菜单。
- **回复设置**：设置关键词自动回复规则。
- **数据统计**：查看 Telegram 渠道的专属销售报表。
