# 项目介绍

**TG 发卡机器人** 是一套基于 **独角数卡** 深度开发的综合解决方案。它以独角数卡作为核心后端，外接一套功能强大的 Telegram 机器人，实现了商品数据、支付方式与订单系统的双向实时同步。

::: tip 核心优势
如果您已经部署了独角数卡，可以直接无缝接入本机器人，瞬间拥有 Telegram 售卖渠道！
:::

### 1. 双端同步，高效管理
- **数据互通**：机器人与网页端商城数据完全同步，无需重复上传商品。
- **双后台管理**：既可以使用独角数卡原生后台，也可以使用机器人专属可视化后台，双管齐下，效率倍增。

### 2. 安全隐秘，无惧攻击
- **隐藏式部署**：利用 Telegram 的特性，您的售卖前端隐藏在机器人中，无需暴露域名。
- **抗 DDOS**：攻击者无法直接攻击您的后端服务器，保障业务连续性。
- **卡密安全**：数据传输加密，杜绝卡密被盗风险。

### 3. 功能强大，体验极佳
- **多支付支持**：原生支持 USDT (TRC20) 等多种加密货币支付。
- **操作便捷**：精心设计的 Telegram 菜单交互，用户上手即用。
- **场景丰富**：支持个人售卖、企业运营以及公群自动发卡等多种场景。
