### TRX 兑换设置
1. 进入后台 -> **兑换管理**。
2. 设置汇率浮动比例（例如：赚取 2% 差价）。
3. 配置资金归集钱包地址。

### 能量租赁配置
1. 进入后台 -> **能量管理**。
2. 添加您的 API 供应商（如：trongas, feiyu 等）。
3. 设置轮询策略，确保高可用。

### 会员充值对接
1. 进入后台 -> **会员管理**。
2. 配置官方会员充值接口密钥。
3. 设置前端售卖价格。
