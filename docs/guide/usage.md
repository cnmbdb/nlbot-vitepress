# 使用手册

本手册将指导您如何作为管理员使用后台，以及用户如何使用 Telegram 机器人。

## 机器人指令 (用户端)

用户与 Telegram 机器人交互时的常用指令：

- `/start` - 启动机器人，查看主菜单
- `购买能量` - 点击菜单按钮，选择时长和数量
- `查询订单` - 查询最近的租赁订单状态
- `个人中心` - 查看余额、推广链接等
- `联系客服` - 获取客服联系方式

## 管理后台 (管理员端)

访问后台地址并登录后，您可以看到以下模块：

### 1. 仪表盘
查看系统概览数据，包括今日订单数、总成交额、新增用户数等。

### 2. 订单管理
- **能量订单**：查看所有用户的能量租赁记录，支持按状态筛选（成功、失败、进行中）。
- **补单操作**：对于失败的订单，可以手动尝试补单或退款。

### 3. 用户管理
- 查看所有使用过机器人的用户列表。
- 修改用户余额（充值/扣款）。
- 封禁违规用户。

### 4. 机器人管理
- **重启机器人**：修改配置后，点击此按钮使更改生效。
- **消息推送**：向所有用户或指定用户群发消息（广播）。

### 5. 能量池对接
- 新建能量池账号：生成 API Key 供外部调用。
- 对接第三方池：配置上游能量池的 API 信息。

## 常见问题

### Q: 机器人没反应怎么办？
A: 
1. 检查 Docker 容器是否正常运行 (`docker compose ps`)。
2. 检查 `bot` 容器日志 (`docker compose logs -f bot`) 看是否有报错。
3. 确认 Token 是否正确，以及服务器网络是否能连接 Telegram API。

### Q: 能量未到账？
A:
1. 检查出款地址是否有足够的 TRX 用于支付手续费（如果需要）。
2. 检查能量池是否有足够的空闲能量。
3. 查看后台订单详情中的错误提示。

::: tip 维护建议
定期备份数据库 (`nl-admin.sql` 或 `/var/lib/mysql` 目录)，以防数据丢失。
:::
