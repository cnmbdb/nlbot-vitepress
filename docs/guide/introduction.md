# 项目介绍

TGNL Admin 是一个基于 Nuxt 3 构建的 Telegram Bot 管理后台系统，提供了完整的机器人管理、用户管理、订单管理等功能。

## 项目架构

### 技术栈

- **前端框架**: Nuxt 3
- **UI 组件库**: Nuxt UI (基于 Tailwind CSS)
- **状态管理**: Pinia
- **数据库**: MySQL 8.0
- **后端**: Nuxt Server API
- **容器化**: Docker & Docker Compose

### 项目结构

```
tgnl-admin/
├── pages/              # 页面组件
│   ├── dashboard.vue   # 仪表盘
│   ├── bots/          # 机器人管理
│   ├── users/         # 用户管理
│   ├── orders/        # 订单管理
│   └── ...
├── server/            # 后端 API
│   └── api/          # API 路由
├── components/        # Vue 组件
├── nl-2333/          # Telegram Bot 代码
│   ├── al.py         # Bot 主程序
│   └── config.txt    # Bot 配置
└── docker-compose.yml # Docker 配置
```
