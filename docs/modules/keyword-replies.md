# 关键词回复

关键词回复模块用于配置自动回复规则，当用户消息包含特定关键词时自动回复。

## 功能特性

### 关键词列表

- 显示所有已配置的关键词
- 关键词和匹配模式
- 回复内容预览
- 状态（启用/禁用）

### 关键词配置

- **关键词** - 要匹配的关键词
- **匹配模式** - 精确匹配/模糊匹配
- **回复内容** - 自动回复的内容
- **回复类型** - 文本/图片/文件等
- **优先级** - 匹配优先级

### 匹配规则

- **精确匹配** - 完全匹配关键词
- **模糊匹配** - 包含关键词即可
- **正则匹配** - 使用正则表达式
- **多关键词** - 支持多个关键词

### 操作功能

- **添加关键词** - 创建新的回复规则
- **编辑关键词** - 修改现有规则
- **删除关键词** - 删除不需要的规则
- **启用/禁用** - 控制规则状态

## 使用说明

### 添加关键词

1. 点击"添加关键词"按钮
2. 输入关键词
3. 选择匹配模式
4. 配置回复内容
5. 设置优先级
6. 保存配置

### 匹配模式

#### 精确匹配
用户消息必须完全等于关键词。

示例：
- 关键词：`帮助`
- 匹配：`帮助` ✅
- 不匹配：`我需要帮助` ❌

#### 模糊匹配
用户消息包含关键词即可。

示例：
- 关键词：`帮助`
- 匹配：`帮助` ✅
- 匹配：`我需要帮助` ✅
- 匹配：`帮助我` ✅

#### 正则匹配
使用正则表达式匹配。

示例：
- 关键词：`/^价格|费用|多少钱/`
- 匹配：`价格是多少` ✅
- 匹配：`费用多少` ✅

## 优先级设置

当多个关键词同时匹配时，优先级高的规则会先执行。

- **高优先级** - 优先匹配
- **中优先级** - 正常匹配
- **低优先级** - 最后匹配

## 回复类型

### 文本回复
纯文本回复，支持 Markdown 格式。

### 图片回复
发送图片，需要提供图片 URL。

### 文件回复
发送文件，支持各种文件类型。

### 组合回复
可以同时发送文本、图片、文件等。

## 最佳实践

1. **避免冲突** - 确保关键词不重复
2. **合理优先级** - 设置合适的优先级
3. **测试验证** - 添加后测试匹配效果
4. **定期维护** - 定期检查和更新规则
