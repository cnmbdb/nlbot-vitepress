# 命令管理

命令管理模块用于配置 Telegram Bot 的命令和响应。

## 功能特性

### 命令列表

- 显示所有已配置的命令
- 命令名称和描述
- 命令状态（启用/禁用）
- 快速操作入口

### 命令配置

- **命令名称** - 设置命令名称（如 /start）
- **命令描述** - 命令的说明文字
- **响应内容** - 命令的回复内容
- **响应类型** - 文本/图片/文件等
- **权限设置** - 命令使用权限

### 命令响应

- 文本回复
- 图片回复
- 文件回复
- 内联键盘
- 自定义响应

### 操作功能

- **添加命令** - 创建新命令
- **编辑命令** - 修改命令配置
- **删除命令** - 删除不需要的命令
- **启用/禁用** - 控制命令状态

## 使用说明

### 添加命令

1. 点击"添加命令"按钮
2. 输入命令名称（以 / 开头）
3. 设置命令描述
4. 配置响应内容
5. 保存命令

### 编辑命令

1. 在命令列表中找到目标命令
2. 点击"编辑"按钮
3. 修改命令配置
4. 保存更改

### 测试命令

1. 在 Telegram 中找到机器人
2. 发送配置的命令
3. 验证响应是否正确
4. 根据需要进行调整

## 命令格式

命令名称必须以 `/` 开头，例如：
- `/start` - 开始命令
- `/help` - 帮助命令
- `/settings` - 设置命令

## 响应类型

### 文本响应

纯文本回复，支持 Markdown 格式。

### 图片响应

发送图片，需要提供图片 URL 或上传图片。

### 文件响应

发送文件，支持各种文件类型。

### 内联键盘

显示内联键盘，用户可点击按钮。

## 权限设置

- **公开** - 所有用户可使用
- **私有** - 仅特定用户可使用
- **管理员** - 仅管理员可使用
