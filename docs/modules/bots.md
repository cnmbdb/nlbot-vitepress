# 机器人管理

机器人管理模块用于管理多个 Telegram Bot，包括配置、监控和状态管理。

## 功能特性

### 机器人列表

- 显示所有已配置的机器人
- 机器人基本信息（名称、Token、状态）
- 快速操作入口

### 机器人配置

- **Token 管理** - 设置和更新 Bot Token
- **基础信息** - 机器人名称、描述等
- **功能开关** - 启用/禁用特定功能
- **通知配置** - 设置通知 URL 和参数

### 状态监控

- 机器人运行状态
- 连接状态检查
- 错误日志查看
- 性能监控

### 操作功能

- **启动/停止** - 控制机器人运行状态
- **重启** - 重启机器人服务
- **日志查看** - 查看机器人运行日志
- **配置编辑** - 编辑机器人配置

## 使用说明

### 添加机器人

1. 点击"添加机器人"按钮
2. 输入 Bot Token（从 @BotFather 获取）
3. 配置机器人基本信息
4. 保存配置

### 编辑配置

1. 在机器人列表中找到目标机器人
2. 点击"编辑"按钮
3. 修改配置项
4. 保存更改

### 查看日志

1. 选择目标机器人
2. 点击"查看日志"
3. 查看实时或历史日志
4. 支持日志搜索和过滤

## 配置说明

### Bot Token

从 Telegram 的 @BotFather 获取：
1. 在 Telegram 中搜索 @BotFather
2. 发送 `/newbot` 命令
3. 按照提示创建机器人
4. 获取 Token

### 通知 URL

配置 Webhook URL，用于接收机器人事件通知。

### 功能开关

- 自动回复
- 命令处理
- 关键词回复
- 其他自定义功能

## 故障排查

### 机器人无法启动

- 检查 Token 是否正确
- 确认网络连接正常
- 查看错误日志

### 连接失败

- 检查 Telegram API 访问
- 确认防火墙设置
- 验证 Token 有效性
