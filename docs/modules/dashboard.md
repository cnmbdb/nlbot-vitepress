# 仪表盘

仪表盘是系统的首页，提供了系统概览和关键数据统计。

## 功能特性

### 数据统计

- **总用户数** - 显示系统中的用户总数
- **机器人数量** - 当前管理的机器人数量
- **订单统计** - 订单总数、今日订单等
- **活跃度** - 用户活跃度统计

### 图表展示

- 用户增长趋势图
- 订单统计图表
- 收入趋势分析
- 其他关键指标可视化

### 实时监控

- 系统运行状态
- 服务健康检查
- 实时数据更新

## 使用说明

1. 登录系统后，默认进入仪表盘页面
2. 查看各项统计数据
3. 通过图表了解数据趋势
4. 监控系统运行状态

## 数据刷新

- 数据自动刷新（默认 30 秒）
- 可手动刷新数据
- 支持自定义刷新间隔
