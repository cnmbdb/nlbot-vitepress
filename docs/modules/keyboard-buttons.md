# 键盘按钮

键盘按钮模块用于管理 Telegram Bot 的键盘按钮配置。

## 功能特性

### 按钮列表

- 显示所有已配置的按钮
- 按钮名称和类型
- 按钮状态（启用/禁用）
- 关联的键盘

### 按钮配置

- **按钮名称** - 按钮的标识名称
- **按钮文字** - 显示在按钮上的文字
- **按钮类型** - 文本/URL/切换等
- **回调数据** - 点击回调数据
- **关联功能** - 按钮关联的功能

### 按钮类型

- **文本按钮** - 发送文本回调
- **URL 按钮** - 打开链接
- **切换按钮** - 状态切换
- **支付按钮** - 支付功能
- **自定义按钮** - 自定义功能

### 操作功能

- **添加按钮** - 创建新按钮
- **编辑按钮** - 修改按钮配置
- **删除按钮** - 删除不需要的按钮
- **复制按钮** - 复制按钮配置

## 使用说明

### 创建按钮

1. 点击"添加按钮"按钮
2. 输入按钮名称
3. 设置按钮文字
4. 选择按钮类型
5. 配置按钮功能
6. 保存按钮

### 按钮类型选择

根据使用场景选择合适的按钮类型：
- **文本按钮** - 用于一般操作
- **URL 按钮** - 用于打开链接
- **切换按钮** - 用于状态切换
- **支付按钮** - 用于支付功能

### 按钮配置

#### 文本按钮
- 按钮文字：显示的文字
- 回调数据：点击后发送的数据
- 回调处理：处理回调的逻辑

#### URL 按钮
- 按钮文字：显示的文字
- URL 地址：打开的链接

#### 切换按钮
- 按钮文字：显示的文字（两种状态）
- 切换逻辑：状态切换的处理

## 按钮管理

### 按钮分组

可以将相关按钮分组管理，方便组织和查找。

### 按钮复用

创建的按钮可以在多个键盘中复用，提高效率。

### 按钮模板

可以创建按钮模板，快速创建相似按钮。

## 使用场景

- **菜单按钮** - 导航菜单
- **操作按钮** - 执行操作
- **选择按钮** - 选择选项
- **链接按钮** - 打开链接
- **支付按钮** - 支付功能

## 最佳实践

1. **命名规范** - 使用清晰的命名
2. **文字简洁** - 按钮文字要简短
3. **功能明确** - 每个按钮功能要明确
4. **合理复用** - 复用常用按钮
5. **测试验证** - 创建后测试效果
