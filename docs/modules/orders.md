# 订单管理

订单管理模块用于查看和处理用户订单。

## 功能特性

### 订单列表

- 显示所有订单信息
- 订单状态（待处理/已完成/已取消）
- 订单金额和详情
- 搜索和筛选功能

### 订单详情

- **基本信息** - 订单号、用户、金额
- **订单状态** - 当前状态和状态历史
- **支付信息** - 支付方式和支付状态
- **操作记录** - 订单操作历史

### 订单处理

- **状态更新** - 更新订单状态
- **订单处理** - 处理订单请求
- **退款处理** - 处理退款申请
- **备注添加** - 添加订单备注

### 数据统计

- 订单总数
- 今日订单
- 订单金额统计
- 订单状态分布

## 使用说明

### 查看订单列表

1. 进入"订单管理"页面
2. 查看订单列表
3. 使用搜索框查找订单
4. 使用筛选器过滤订单

### 处理订单

1. 选择要处理的订单
2. 查看订单详情
3. 更新订单状态
4. 添加处理备注

### 订单筛选

- **状态筛选** - 按订单状态筛选
- **时间范围** - 按创建时间筛选
- **金额范围** - 按订单金额筛选
- **用户筛选** - 按用户筛选

## 订单状态

- **待处理** - 订单已创建，等待处理
- **处理中** - 订单正在处理
- **已完成** - 订单已完成
- **已取消** - 订单已取消
- **已退款** - 订单已退款

## 订单信息字段

- **订单号** - 唯一订单标识
- **用户信息** - 下单用户
- **订单金额** - 订单总金额
- **创建时间** - 订单创建时间
- **支付时间** - 支付完成时间
- **完成时间** - 订单完成时间
- **订单状态** - 当前状态
- **备注信息** - 订单备注
