# 用户管理

用户管理模块用于查看和管理 Telegram Bot 的用户信息。

## 功能特性

### 用户列表

- 显示所有用户信息
- 用户基本信息（ID、用户名、注册时间）
- 用户状态（活跃/非活跃）
- 搜索和筛选功能

### 用户详情

- **基本信息** - 用户 ID、用户名、昵称
- **统计数据** - 消息数、订单数、消费金额
- **操作记录** - 用户操作历史
- **关联数据** - 订单、充值记录等

### 数据统计

- 总用户数
- 活跃用户数
- 新增用户趋势
- 用户分布统计

### 操作功能

- **查看详情** - 查看用户完整信息
- **数据导出** - 导出用户数据
- **用户搜索** - 按 ID、用户名搜索
- **筛选过滤** - 按状态、时间筛选

## 使用说明

### 查看用户列表

1. 进入"用户管理"页面
2. 查看用户列表
3. 使用搜索框查找特定用户
4. 使用筛选器过滤用户

### 查看用户详情

1. 点击用户列表中的用户
2. 查看用户详细信息
3. 查看统计数据
4. 查看操作记录

### 数据导出

1. 选择要导出的用户（或全部）
2. 点击"导出"按钮
3. 选择导出格式（CSV/Excel）
4. 下载导出文件

## 用户信息字段

- **用户 ID** - Telegram 用户唯一标识
- **用户名** - @username
- **昵称** - 显示名称
- **注册时间** - 首次使用时间
- **最后活跃** - 最后活动时间
- **消息数** - 发送消息总数
- **订单数** - 订单总数
- **余额** - 账户余额

## 筛选选项

- **状态筛选** - 活跃/非活跃
- **时间范围** - 注册时间、活跃时间
- **用户类型** - 普通用户/VIP 等
- **自定义筛选** - 按其他条件筛选
