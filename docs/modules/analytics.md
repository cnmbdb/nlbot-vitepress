# 数据分析

数据分析模块提供详细的数据统计和分析功能。

## 功能特性

### 数据统计

- **用户统计** - 用户增长、活跃度分析
- **订单统计** - 订单数量、金额分析
- **收入统计** - 收入趋势、收入分布
- **机器人统计** - 机器人使用情况

### 图表展示

- 折线图 - 趋势分析
- 柱状图 - 数据对比
- 饼图 - 数据分布
- 其他可视化图表

### 报表功能

- 数据报表生成
- 报表导出（PDF/Excel）
- 自定义报表
- 定时报表

### 数据筛选

- 时间范围选择
- 多维度筛选
- 自定义查询
- 数据对比

## 使用说明

### 查看统计数据

1. 进入"数据分析"页面
2. 选择要查看的数据类型
3. 选择时间范围
4. 查看统计结果和图表

### 生成报表

1. 选择报表类型
2. 设置筛选条件
3. 选择时间范围
4. 生成并导出报表

### 数据对比

1. 选择对比维度
2. 选择对比时间段
3. 查看对比结果
4. 分析差异原因

## 统计维度

### 时间维度

- 按日统计
- 按周统计
- 按月统计
- 按年统计
- 自定义时间范围

### 数据维度

- 用户数据
- 订单数据
- 收入数据
- 机器人数据
- 其他业务数据

## 图表类型

- **趋势图** - 显示数据随时间的变化趋势
- **对比图** - 对比不同时间段或不同维度的数据
- **分布图** - 显示数据的分布情况
- **占比图** - 显示各部分占比

## 报表导出

支持的导出格式：
- PDF 格式
- Excel 格式
- CSV 格式
